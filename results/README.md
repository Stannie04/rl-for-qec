# The `results/` directory

This directory contains the results of the experiments, including saved model checkpoints, datasets of generated shots, and analysis results. Note that subdirectories are not created automatically (except for `patterns/`, instead results have been moved manually for better organization. The main components of this directory are as follows:

- `n_k_d_[ldpc|toric]/`: Contains the results for the specific code configuration, including plots, saved models, and datasets.
- `patterns/`: Contains examples of generalized error patterns that can be used for training and evaluation. These can be generated by running the 'datasets' experiment to extract decoder mistakes, then running 'analysis' to generate the patterns.
- `mistake_distributions/`: Contains frequency plots of generalized error patterns that different decoders make mistakes on. These plots can are generated using the method above as well.
