# Thesis Notes: Flexible GNN Decoders for Arbitrary Tanner Graphs

### 0. Funnel

Quantum information is fragile, no chance to get there with physical qubit, so we need redudancy and stabilization. Those are error codes, but they need to scale. First demonstrations were using surface code, where error decoding is local allowing to use local heuristic decoders. But if we want to have useful QC we need to scale and be more efficient. This is possible with LDPC codes, but they need complex and fast decoders. There is a progress with neural decoders, but they are specialised agent to deal with only one code. In reality, the two-qubit logical operation might require lattice surgery which locally modifies the code structure, so we need to be able to adapt the decoder to different code structures. We want to investigate how general a decoder can be, and how it can be adapted to different code structures.

### 4. Vibe-stormed Observations & Hints

* **The Routing Advantage:** A mixture-of-decoders model is essentially a smart router. It can instantly send sparse, "easy" errors to fast deterministic algorithms (like Union-Find) and reserve the computationally heavy GNN strictly for highly ambiguous, dense syndromes, keeping average latency low.
* **The Hardware Reality (Correlated Noise):** Static decoders like MWPM struggle with non-Markovian or strongly biased noise. A GNN implicitly learns the real noise topology of the chip, excelling where crosstalk or $Z$-dominated errors warp the expected distributions.
* **Dynamic Code Switching:** Lattice surgery deforms the Tanner graph boundaries in real-time. Standard decoders need an algorithmic reset to handle this; a universal GNN agent can decode continuously across these transitioning boundaries.
* **Software-Only Pauli Tracking:** Tracking the physical Pauli frame purely in software prevents the system from injecting real hardware errors via noisy correction pulses, effectively giving the agent a continuous memory (Bayesian prior) of the system state.
* **The Degeneracy Trap:** Stabilizer degeneracy is the main hurdle for physical tracking. If the agent tries to track exact physical paths during severe error cascades, its confidence will collapse due to indistinguishable equivalent error strings.
* **The Master Synergy:** The true power of the mixture model is switching philosophies dynamically. The agent uses physical tracking during sparse regimes to maintain a granular memory, but detects when "syndrome frustration" is high and falls back to logical tracking (cosets) to avoid hallucinating incorrect physical states.
