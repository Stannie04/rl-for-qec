# Near-term Roadmap
- Fine-tune the environment through HPO and more elaborate reward structures.
- Implement and benchmark additional RL algorithms (e.g., A3C, SAC).
- Evaluate the performance of different types of multivariate bicycle codes.
- Evaluate the use of GNNs as function approximators for RL agents in this context.


# Long-term ideas
- Implement curriculum learning strategies to progressively train agents on more complex codes.
- Investigate transfer learning to adapt agents trained on smaller codes to larger ones.
- Make the training loop fully JAX-compatible for performance optimization.


# Open Questions (Topic interest, not implementation-specific)
- How does the geometric definition of bicycle codes relate to algebraic definitions?
- How can we infer the pattern of logical operators from the geometric structure of the codes?
